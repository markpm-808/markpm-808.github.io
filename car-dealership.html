Car Dealership

  • Category: Full Stack Application
  • Project date: 06 June, 2021
  • Source Code: View

Project Description:

Full-stack web application that models a simplified version of a car dealership’s website and sales support system. Includes a public-facing area where users can browse new and used vehicles, view the current specials going on, and contact the dealership. There is an area for sales staff to record customer information for a purchase and an admin area that is used to manage users, inventory, specials, reports, and application specifics. Application set up to use Microsoft-SQL Server with the help of SQL scripts to build database, create tables, and run stored procedures.
